# Proyecto Tracker de Medios

Una aplicación en Tkinter que permite gestionar y seguir el progreso de películas, libros y series, incluyendo funcionalidades de CRUD (Crear, Leer, Actualizar, Eliminar) y seguimiento de calificaciones y reseñas.

## Características

- Ver registros (Películas, Libros, Series)
- Agregar, editar y eliminar registros
- Filtrar por estado (Pendiente, En curso, Terminada)
- Calificación y reseña por cada medio
- Interfaz gráfica amigable con Tkinter

## Tecnologías

- Python
- Tkinter
- SQLite (para la base de datos)

## Instalación

1. Clona el repositorio:
    ```bash
    git clone https://github.com/TU_USUARIO/TU_REPO.git
    ```

2. Instala las dependencias:
    ```bash
    pip install -r requirements.txt
    ```

## Uso

1. Ejecuta el archivo principal:
    ```bash
    python main.py
    ```

2. Interactúa con la interfaz gráfica para agregar, actualizar y eliminar medios.
